# Consulta de CEP

Projeto desenvolvido utilizando JavaScript, Fetch API (AJAX) e a API pública ViaCEP para realizar consultas de CEP e exibir informações de endereço.

## Funcionalidades

* Consulta de CEP em tempo real
* Consumo de API pública ViaCEP
* Exibição de:

  * Logradouro
  * Bairro
  * Cidade
  * Estado
* Interface responsiva para desktop e dispositivos móveis
* Tratamento de erros para CEP inválido ou inexistente

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* ViaCEP API

## Como Executar

1. Clone ou faça download deste repositório.
2. Abra o arquivo `index.html` em seu navegador.

Ou acesse a versão online:
